# CSE120SP23 Project

This is the starter code of the nachos project for CSE120SP23 at UCSD. For an overview 
of nachos, see [nachos/README](nachos/README)

## Important Information

Do not change the directory structure from

```bash
.
├── README.md
└── nachos
    ├── Makefile
    ├── README
    ├── ag
    ├── bin
    ├── machine
    ├── network
    ├── proj0
    ├── proj1
    ├── proj2
    ├── proj3
    ├── security
    ├── test
    ├── threads
    ├── userprog
    └── vm
```

to 

```bash
.
```
